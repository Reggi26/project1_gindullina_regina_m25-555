# Лабиринт Сокровищ

Текстовая приключенческая игра, где вы исследуете таинственный лабиринт, решаете головоломки и ищете легендарные сокровища.

## Запуск игры

### Установка и запуск с Poetry:

poetry install  
poetry run python main.py

### Альтернативный способ (Makefile):

make install  
make run

## Управление
Вот все команды, которые можно использовать в игре:

 - north, south, east, west - Идти в направлении (можно без слова go)

 - look - Осмотреть комнату вокруг

 - take <предмет> - Взять предмет

 - use <предмет> - Использовать предмет

 - inventory - Посмотреть что в рюкзаке

 - solve - Решить загадку в комнате

 - open - Открыть сундук (в комнате сокровищ)

 - show_help - Показать все команды

 - quit - Выйти из игры

Совет: Для движения можно писать просто north, без слова go!

## Локации
В лабиринте 8 уникальных комнат:

 - Вход (entrance) - Начало пути, здесь лежит старый факел

 - Зал (hall) - Большой зал с эхом и пьедесталом

 - Комната ловушек (trap_room) - Опасная комната с хитрой плиткой

 - Библиотека (library) - Пыльная комната со старыми свитками

 - Оружейная (armory) - Комната с оружием и шкатулками

 - Комната сокровищ (treasure_room) - Финальная цель с большим сундуком

 - Тайная комната (secret_room) - Секретная комната с древними фресками

 - Сад (garden) - Заброшенный сад с фонтаном

## Цель игры
Ваша главная задача: найти и открыть сундук с сокровищами!

Для этого нужно:

 - Исследовать все комнаты лабиринта

 - Собрать нужные предметы (ключи, факел, меч)

 - Решить все загадки на своем пути

 - Найти ключ от комнаты сокровищ

 - Открыть сундук и получить победу!

## Подсказки для успеха
 - Факел (torch) - очень важен в темных комнатах!

 - Меч (sword) - поможет в опасных ситуациях

 - Ключи - сохраняйте все найденные ключи

 - Внимательно читайте описания комнат - там есть подсказки

 - Проверяйте инвентарь перед важными решениями
